# 重构智能 ReAgent · AI 自动化营销系统

对标"小猫AI"的新一代 AI 自动化营销平台。

## 核心功能
| 模块 | 功能 | 状态 |
|------|------|------|
| 🎬 智能短视频 | 自动剪辑配音字幕，一键分发多平台 | ✅ 已上线 |
| 💬 AI 客服 | 微信客服机器人，多轮对话 | ✅ 已上线 |
| 🎯 线索筛选 | AI 自动评分、分类 | ✅ 已上线 |
| 📄 AI 文员 | 文档自动生成 | 🚧 开发中 |
| 🤖 AI 编程 | 代码辅助生成 | 🚧 开发中 |

## 快速开始
```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
